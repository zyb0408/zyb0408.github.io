

Prompt Vault 隐私政策

生效日期： [2025年5月29日]

引言
欢迎使用 Prompt Vault！本应用由 [你的名字或你的开发者名称]（以下简称“我们”）开发。我们非常重视您的隐私。本隐私政策旨在说明当您使用 Prompt Vault 应用时，我们如何处理您的信息。

我们收集的信息
Prompt Vault 帮助您存储和管理您的文本提示词（包括名称、类别、标签和内容）以及API密钥（包括名称、平台和密钥内容）。
您在 Prompt Vault 中输入的所有数据，包括您的提示词和API密钥，都完全存储在您的本地设备上，位于应用的私有存储空间内。作为开发者，我们无法访问这些数据，这些数据也不会被传输到任何服务器或任何第三方。
本应用还会将您的主题偏好和语言设置存储在您的设备本地，以便为您提供个性化的使用体验。

我们如何使用您的信息
您存储在 Prompt Vault 中的信息仅用于提供应用的核心功能：即帮助您按照您的意图组织、管理、搜索和访问您的提示词与API密钥。
您的主题和语言偏好用于根据您的选择来显示应用界面。

数据存储与安全
您的所有数据（提示词、API密钥、偏好设置）都使用 Apple 的 SwiftData 框架存储在您设备的本地，并处于应用的沙盒环境中。这些数据的安全性依赖于您 iOS 设备自身的安全特性（例如，设备密码、面容ID、触控ID）。
除了操作系统为应用数据提供的安全保护外，我们没有在本应用中对本地数据实施额外的加密层。

数据共享与第三方
Prompt Vault 不会与任何第三方共享您的任何个人数据或应用内容（提示词、API密钥）。
本应用不使用任何第三方分析服务、广告服务，也没有外部服务器来存储您的数据。

您对信息的控制权
您对自己输入到 Prompt Vault 中的数据拥有完全的控制权。您可以随时在应用内直接添加、查看、编辑和删除您的提示词与API密钥。
如果您从设备中删除本应用，其本地存储的数据通常也会随之被移除（具体取决于 iOS 的数据管理机制和您是否对设备进行了备份）。

儿童隐私
Prompt Vault 不适用于4岁以下（或您所在司法管辖区的同等最低年龄）的儿童。我们不会故意收集儿童的个人信息。

本隐私政策的变更
我们可能会不时更新本隐私政策。如有任何变更，我们会在此页面上发布新的隐私政策来通知您。建议您定期查看本隐私政策以了解任何更改。
